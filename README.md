# 🎬 کلون فرانت‌اند دیزنی پلاس | Disney+ UI Clone

یک پروژه تمرینی و جذاب برای بازطراحی و پیاده‌سازی فرانت‌اند وب‌سایت محبوب **دیزنی پلاس (Disney+)** با تمرکز روی جزییات، انیمیشن‌های CSS و طراحی کاملاً واکنش‌گرا (Responsive).

---

## 🚀 ویژگی‌های کلیدی پروژه

* **📱 کاملاً واکنش‌گرا (Responsive):** بهینه‌سازی شده برای نمایش بی‌نقص در انواع نمایشگرها (موبایل، تبلت و دسکتاپ) با استفاده از Bootstrap 5.
* **⚡ منوهای تعاملی و پویا:** پیاده‌سازی منوهای کشویی (Dropdown) در هدر با استفاده از جاوااسکریپت خالص (Vanilla JS) بدون نیاز به کتابخانه‌های سنگین.
* **🎠 اسلایدر کاستومایز شده:** استفاده از کاروسل بوت‌استرپ و بهینه‌سازی ابعاد تصاویر با ویژگی `object-fit: contain` برای جلوگیری از دفرمه شدن عکس‌ها.
* **✨ انیمیشن‌های نرم CSS:** هاور افکت‌های جذاب روی باکس تصاویر و لوگوها برای پویایی بیشتر رابط کاربری.

## 🛠 ابزارها و تکنولوژی‌های استفاده شده

در این پروژه از هیچ فریم‌ورک جاوااسکریپتی استفاده نشده تا مهارت‌های پایه فرانت‌اند به چالش کشیده شود:

* **HTML5:** ساختاردهی استاندارد و معنایی صفحات.
* **CSS3:** استایل‌دهی اختصاصی، مدیریت انیمیشن‌ها و پوزیشن‌ها.
* **Bootstrap 5:** پیاده‌سازی سریع سیستم گراید (Grid) و کامپوننت کاروسل (Carousel).
* **JavaScript (Vanilla):** مدیریت رویدادهای `mousemove` و `mouseleave` برای انیمیشن‌ها و منوها.
